Here are some examples:

myremote=http://192.168.6.3/foldermyremote=/folder/subfoldermyremote=c:/folder/subfolder uri-or-path : corresponds the given uri. will be named uri's last path component name. Here are some examples:

http://192.168.6.3/folder : will be named 'folder'myremote=/folder/subfolder : will be named 'folder'myremote=c:/folder/subfolder : will be named 'folder'
